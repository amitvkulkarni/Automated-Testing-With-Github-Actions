# Automated-Testing-With-Github-Actions
Exploring Pytest and GitHub Actions to automate the data validation process 

[Read more on my blog:](https://www.analyticsvidhya.com/blog/2021/04/github-workflows-for-test-automation/)
The objective of the blog was to showcase features of Pytest / GitHub actions / vscode and how easy it is to automate many of the routine data-related activities that are carried out day in day out. One can also use a more sophisticated cloud platform with advanced features which let you achieve similar results with automation but, if you have a smaller team, a limited budget, and a shortage of test automation skills then Pytest / GitHub is more than handy to accomplish your project objectives.
